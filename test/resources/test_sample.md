# どんなツール?
**aviutlのオブジェクトファイル(テンプレ用)** とmarkdownから以下を生成する
1. ttsで読み上げる内容の音声
2. 動画に表示するスライド
3. 1,2を含むaviutlのオブジェクトファイル
